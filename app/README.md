# TravelMate — Mini App (прототип фронта)

Кликабельный прототип Telegram Mini App в дизайн-языке проекта: 4 палитры,
переиспользуемые компоненты, роутинг, мок-данные.

## Запуск

```bash
cd app
npm install
npm run dev
```

Открой ссылку из консоли (обычно http://localhost:5173). На десктопе экран
показан в рамке телефона; на телефоне — на весь экран.

## Что можно пощупать

Нижняя навигация: **Сегодня · Поездки · Расходы · Фото · Профиль**. Почти всё
кликается и переходит. Реализованы экраны (≈25):

- **Поездки:** список, создание поездки, приглашение (share-ссылка), join по инвайту.
- **Поездка:** табы Обзор / Активности / Расходы / Фото / Итоги.
- **Активности:** детали (голосование, участники, комментарии), создание, завершение.
- **Расходы:** создание (режимы разделения), детали/разделение, чек+OCR, баланс,
  подтверждение перевода.
- **Фото:** момент (N ракурсов), загрузка, photo-prompt, bingo.
- **Wrapped:** групповой и личный (история-слайды с тапом).
- **Профиль:** выбор палитры (мгновенно + localStorage), реквизиты, уведомления,
  здоровье.

Экраны-демо без перехода из меню: `/splash`, `/prompt` — открой по прямому адресу.

## Структура

```
src/
├ theme.tsx        — провайдер палитры + переключатель
├ data.ts          — мок-данные (поездка, активности, расходы, баланс…)
├ ui.tsx           — общие компоненты (BottomNav, иконки, примитивы, рамка)
├ index.css        — токены 4 палитр + классы компонентов (+ Tailwind)
├ App.tsx          — роутинг и каркас
└ screens/         — экраны
```
